# Cyber Aurora 壁纸皮肤使用教程

> 这是一个非官方的运行时皮肤：它只在当前启动的窗口中叠加背景图和科技风样式，不修改 ChatGPT/Codex 安装包，也不会读取、备份或写入 `~/.codex/config.toml`。

## 仓库中包含什么

| 位置 | 内容 |
| --- | --- |
| `assets/backgrounds/aurora-night.png` | 默认极光夜景壁纸 |
| `assets/cyber-skin.css` | 科技风、文字可读性与控件样式 |
| `assets/renderer-inject.js` | 将背景图传给当前窗口的渲染代码 |
| `scripts/start-cyber-skin.ps1` | 启动带壁纸的 ChatGPT/Codex |
| `scripts/set-cyber-background.ps1` | 选择自己的背景图片 |
| `scripts/restore-cyber-skin.ps1` | 从当前窗口移除皮肤 |
| `scripts/launch-cyber-aurora.ps1` | 供任务栏一键启动器调用 |
| `scripts/injector.mjs` | 本机回环调试连接与样式注入器 |

## 首次使用与每次启动

1. 确认平时直接打开 ChatGPT/Codex 是正常的。
2. **自己关闭所有** ChatGPT/Codex 窗口。
3. 在本文件所在的 `windows` 目录打开 PowerShell，执行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\start-cyber-skin.ps1
```

4. 等待 ChatGPT/Codex 新窗口打开。看到极光背景、深色侧栏和高对比度文字即表示成功。

这个启动方式只在本机回环地址 `127.0.0.1:9336` 开放调试端口，不会对局域网开放端口。

## 任务栏一键启动

如果已经将 `Cyber Aurora.lnk` 固定到任务栏，之后直接点击它即可。

若点图标时普通 ChatGPT/Codex 仍开着，启动器只会提示你先关闭窗口；它不会强制结束进程，也不会修改配置。

## 更换自己的壁纸

皮肤窗口已经打开时，在 `windows` 目录运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\set-cyber-background.ps1
```

随后在弹出的选择框中选 PNG、JPG、JPEG 或 WebP 图片。单张图片限制为 15 MB；选择后会立即应用。你的原图不会被修改，皮肤只会复制一份到 `assets/backgrounds/`。

也可以直接指定图片路径：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\set-cyber-background.ps1 -Path "F:\壁纸\你的图片.png"
```

## 停用或恢复普通外观

运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\restore-cyber-skin.ps1
```

它只停止当前皮肤的注入器并尝试移除当前窗口中的样式；不会删除主题、聊天记录、账号数据或 `.codex` 配置。之后关闭该窗口，再按平时方式启动 ChatGPT/Codex 即可。

## 常见情况

### 直接点普通 ChatGPT 后没有壁纸

这是正常的。壁纸皮肤不会永久嵌入普通启动流程，以避免影响商店版更新、应用签名或你的 Codex 配置。请从 Cyber Aurora 启动器进入壁纸版。

### 打开后文字看不清

请确认仓库版本至少包含 `Improve Cyber Aurora text contrast` 提交；该版本已针对侧栏、正文、输入框与浅色功能卡片做了独立的可读性处理。

### 启动器提示先关闭窗口

请手动关闭当前所有 ChatGPT/Codex 窗口后重试。启动器刻意不会替你强制结束任何应用。

## 不会做的事情

- 不修改 `WindowsApps`、`app.asar` 或应用签名。
- 不修改 `~/.codex/config.toml`。
- 不自动创建桌面图标或开始菜单图标。
- 不上传你的壁纸到网络。
